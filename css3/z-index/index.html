CSS3 - Z-index

La propiedad z-index indica el z-order (orden en el eje z) de un elemento y sus descendientes.

Cuando varios elementos se superponen, z-order determina cuales cubren a otros.

Un elemento con mayor z-index generalmente cubre a otro menor.

Solo se aplica sobre elementos "posicionados".

z-index: auto;
