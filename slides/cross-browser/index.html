Vendor Prefixes

Los browsers tienen diferentes motores de renderizado de los sitios HTML.

Algunos features de CSS3 son implementados de distinta forma y por eso les agregan a las properties un prefijo.


.prefixed-class {
    display: -webkit-box;
    display: -webkit-flex;
    display: -ms-flexbox;
    display: flex;
    background: -webkit-linear-gradient(red, green);
    background: linear-gradient(red, green);
}
