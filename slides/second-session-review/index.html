Componentes UI

Navbar

Se útiliza como herramienta principal para navegar el sitio, normalmente esta visible y accesible todo el tiempo para el usuario

Sidebar

Se usa para poner acciones secundarias de navegación o manipulación de contenido, como filtros. Tambien puede ser usada a modo de Navbar.

Footer

Siempre se encuentra a lo último de la web, normalmente contiene información institucional del sitio y links a diversas secciones del mismo.

...

Breadcrums

Sirve para marcar un historial de navegación, dandole contexto al usuario de en que parte de la aplicación se encuentra.

Modal

Es un panel que aparece por delante de todo el contenido, cortando el flujo de navegación, es un componente invasivo que debe ser usado con precaución

Tabs

Sirve para organizar el contenido fragmentandolo, facilitando la lectura del mismo y evitando la sobrecarga de información
